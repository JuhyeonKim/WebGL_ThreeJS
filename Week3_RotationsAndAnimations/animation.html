1. Animation
    • 이것만은 기억하자
      • Linear interpolation: 선형 보간법
        • T초 동안 초기값 P0에서 최종값 P1으로 변화하는 애니메이션, 애니메이션이 n초 친행 되었을때
        • t = n/T, 따라서 t = [0, 1]
        • f(t) = P0*(1 - t) + P1*t, 원하는 성분(x, y, z, opacity, scale, 등)에 대해서 개별적으로 적용
      • Parametric equation: 함수를 다른 매개변수를 사용해 나타냄
        • T초 동안 어떤 값 P0가 P1으로의 변화가 것을 매개변수 t 기준으로 수식으로 정의 되었을때, 애니메이션이 n초 진행 되었을때
        • t = n/T, 따라서 t = [0, 1]
        • 상대적 변화로 기술: f(t) = P0 + (t에 대한 다항식)
        • 절대적 값으로 변화 기술: f(t) = (t에 대한 다항식)
      • 매개변수 t에 대한 함수 적용을 통해 역동적인 애니메이션 구현
        • 참고
        • accelerate
        • decelerate
        • overshoot
        • bounce
