1. 색상이론
    • 이것만은 기억하자
      • 사용 목적: Fragment shader 내에서 색상 합성할 때(lighting model), 렌더링시에 블렌딩이 이루어질때
      • 3원색
        • 가산혼합(additive color): RGB ✓
        • 감산혼합(subtractive color): CMYK
      • 다양한 색상 모델
        • RGB
        • HSL (Hue, Saturation, Lightness)
        • HSV (Hue, Saturation, Value)
      • 다양한 색상 포맷
        • RGB, RGBA (사용 비트수 다양): 비트맵에 사용
        • yuv: 동영상에 많이 사용
        • YCbCr: 옛날 영상 기기
      • OpenGL에서
        • 주로 RGB 계열 사용
        • 하지만 텍스쳐로 사용하는 이미지의 경우에는 당양한 포맷을 접하게 될수도 있음
      • Lighting
        • 최종 색 = 텍스쳐 색 + ambient + directional + specular
        • directional과 specular를 위해서는 vertex normal attribute 있어야함(보는 각도에 따라 반사되는 빛이 다르니까)
        • 가산 혼합이므로 어떤 종류의 빛이든지 상관 없이 합하면 합할수록 밝아짐(흰색)
      • Blending: Source(픽셀에 칠하려는 색깔)에 Dest(픽셀의 현재 색깔)을 입히는 방식
        • Alpha 채널을 전혀 사용하지 않으면 고려할 필요 없고, 퍼포먼스를 위해서 알파 채널을 사용하지 않기도 함
        • 블렌딩 활성화: glEnable(GL_BLEND)
        • 블랜딩 방식에 따라 여러 공식이 존재
        • 우리가 기대하는 보통의 블렌딩: glBlendFunc(GL_SRC_ALPHA, GL_ONE_MINUS_SRC_ALPHA)
